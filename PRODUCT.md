# Product

## Register

product

## Users
Frontend developers and UI designers visually editing React, Vue, or Svelte apps in their local development environment and writing styling and layout changes directly back to their codebase.

## Product Purpose
Glide is a local visual design tool that runs on top of a running frontend application, offering a Figma-like workspace to edit styles and layout with direct AST write-back to source code. It eliminates the need for proprietary cloud formats, providing developer-driven local editing with zero lock-in.

## Brand Personality
Editorial, confident, and playful yet precise. Employs Figma-like contrast (monochrome frames, bold pastel color blocks, and figmaSans type) to combine developer precision with a joyful design aesthetic.

## Anti-references
Typical SaaS templates with heavy shadows, gradient card borders, low-contrast gray text on tinted white surfaces, retro monospace console clunkiness, and overly complex, high-density toolbars.

## Design Principles
1. **Source Code Integrity First**: The developer's codebase is the source of truth. Transformations must be minimal, clean, and write back structure without adding boilerplate or disrupting developer styling choices.
2. **Predictable Precision**: Actions like dragging, resizing, and snapping must behave exactly as predicted, using Kd-Trees and smart guides to align perfectly on whole-number pixel grids or neighboring elements.
3. **Clean Chrome, Rich Canvas**: The editor chrome (toolbar, properties panels, and layers tree) must act as a clear, high-contrast frame, allowing the user's application on the canvas to remain the absolute focus.

## Accessibility & Inclusion
Strict adherence to WCAG AA contrast (>= 4.5:1) for all interactive options and text. Clear, distinct keyboard focus states, support for keyboard shortcuts, and full support for reduced-motion options (using crossfades or instant transitions).
